JSinject Documentation

Injector

JSinject features an injector for JavaScript code. It is only recommended for experienced developers, but if you have a JavaScript file (currently online only) you can fetch it with this simple code:
fetch("The URL").then((res) => res.text()).then((js) => eval(js));
Replace "The URL" with the .js. Usually, with GitHub, the URL should be something like https://raw.githubusercontent.com/us....file.js. If your file is like this, congratulations!

Troubleshooting

If nothing appears when the about:blank window opens, or the window does not open, try these fixes:

  1. Redirects/New windows not allowed by github.io or disnos9.github.io.
  2. Your code is not formatted correctly. Learn more about using Injector.
  3. We were unable to open an about:blank window.

This will only appear if the about:blank window fails to appear.

Hotlinks

12/21/2023 - HOTLINKS IS OFFICIALLY OUT! I finished the code and Hotlinks is now available for use. You can find IDs and more in the README file in our repo or our JSON file with all the IDs.


Hotlinks is a completely new feature designed by disnos9 that changes everything with JavaScript Injection and Injection Scripts. A small tag will be added to the end of the base URL, and when this page detects that tag, it will inject the Hotlinks ID's JavaScript code into a new about:blank window, and then close the tab. (Not the about:blank, the referrer tab) Here's some documentation about how it works.
Upon entering your target page, you will get an alert saying that the website you are visiting is a JSinject Hotlinks URL, with a link to this documentation.

Hotlinks ID

Every website (or Injection Script) is assigned an ID. When this ID is called for, that script will be called for.

Troubleshooting

If you recieve an error when you go to a Hotlinks ID, you will be referred to this (Troubleshooting). You got this error because:

  1. The Hotlinks ID is incorrect or invalid.
  2. The JavaScript code is not functioning properly or broke.
  3. (If it worked previously) The Hotlinks ID was removed.

URL

The URL should look something like this:

https://disnos9.github.io/jsinject/?id=320422
If it does not, it is not a valid Hotlinks ID URL.
